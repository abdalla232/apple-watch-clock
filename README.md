⌚ Apple Watch Style Analog Clock (Python GUI)

A modern, responsive analog clock built with Python using Tkinter. The project simulates an Apple Watch-inspired interface with smooth animation, real-time updates, and a scalable design that adapts to window resizing.

✨ Features
🕒 Real-time analog clock with smooth movement
📱 Responsive design (auto-centers and resizes with window)
🎨 Apple Watch-inspired minimal UI
⚡ 60 FPS animation using continuous updates
🔴 Moving second, minute, and hour hands
🖥️ Digital time display
🖱️ Zoom support (mouse wheel / controls)
📐 Fully dynamic scaling based on window size
🧠 Technologies Used
Python 3
Tkinter (GUI)
Math module for angle calculations
Datetime for real-time clock updates
🚀 How to Run
pip install tk
python clock.py

Note: Tkinter usually comes pre-installed with Python.

🎯 Purpose

This project was created to practice GUI development in Python, real-time animation logic, and mathematical transformations for rendering analog systems.
